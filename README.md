## input
COSMAC VIP this layout: <br>
1 2 3 C <br>
4 5 6 D <br>
7 8 9 E <br>
A 0 B F <br>
so thats just mapped to: <br>
1 2 3 4 <br>
q w e r <br>
a s d f <br>
z x c v <br>
