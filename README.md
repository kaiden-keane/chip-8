# silly little interpreter for funsies!

## input
this interpreter maps COSMAC VIP layout to a modern layout as follows:
<table>
<tr>
<td><b>COSMAC VIP layout</td>
<td><b>keyboard layout</td>
</tr>

<tr>
<td>
1 2 3 C <br>
4 5 6 D <br>
7 8 9 E <br>
A 0 B F <br>
</td>
<td>
1 2 3 4 <br>
q w e r <br>
a s d f <br>
z x c v <br>
</td>
</tr>
</table>
